Teoria

Equazioni del moto

Sistema a 2 gradi di libertà (θ₁, θ₂) dalla verticale discendente. Lagrangiana ℒ = T − V con:

T = ½(m₁+m₂)L₁²ω₁² + ½m₂L₂²ω₂² + m₂L₁L₂ω₁ω₂cos(θ₁−θ₂)
V = −(m₁+m₂)gL₁cosθ₁ − m₂gL₂cosθ₂

In forma matriciale M·θ̈ = f, con det M = m₂L₁²L₂²[m₁+m₂sin²(Δθ)] > 0:

M = [(m₁+m₂)L₁², m₂L₁L₂cosΔθ] f₁ = −m₂L₁L₂ω₂²sinΔθ − (m₁+m₂)gL₁sinθ₁
[m₂L₁L₂cosΔθ, m₂L₂² ] f₂ = +m₂L₁L₂ω₁²sinΔθ − m₂gL₂sinθ₂
Conservazione dell'energia: E = T + V è un invariante del moto. La deriva numerica con RK4 (h = 4 ms) è tipicamente < 0.1% su 30 s.

Teoria

Il sistema di Lorenz

ẋ = σ(y − x) /* σ: numero di Prandtl */
ẏ = x(ρ − z) − y /* ρ: numero di Rayleigh ridotto */
ż = xy − βz /* β = 8/3 nell'originale */

Dissipazione e attrattore strano

∇·F = −(σ+1+β) = −13.67 < 0: il volume nello spazio delle fasi si contrae esponenzialmente. Le traiettorie convergono sull'attrattore frattale (D_H ≈ 2.06) con λ₁ ≈ 0.9 s⁻¹. I tre punti fissi C₀=(0,0,0) e C±=(±√(β(ρ−1)), ±√(β(ρ−1)), ρ−1) sono tutti instabili per σ=10, ρ=28, β=8/3.

Effetto farfalla: Lorenz scoprì il caos nel 1961 re-eseguendo una simulazione con valore arrotondato (0.506 invece di 0.506127). Dopo pochi minuti simulati la traiettoria era completamente divergente dall'originale.

Il sistema di Rössler

ẋ = −y − z
ẏ = x + ay
ż = b + z(x − c) /* unico termine non lineare */

Cascata di Feigenbaum (a=b=0.2):
c ≈ 2.0 → p.1   c ≈ 3.5 → p.2   c ≈ 4.2 → p.4   c ≈ 5.7 → caos
Costante di Feigenbaum: δ_F = lim (rₙ−rₙ₋₁)/(rₙ₊₁−rₙ) = 4.6692... — universale per tutte le mappe unimodali con massimo quadratico.

La mappa logistica

xₙ₊₁ = r · xₙ · (1 − xₙ), r ∈ [0,4], xₙ ∈ [0,1]

r ≤ 1 → estinzione r_∞ ≈ 3.5699 → inizio caos
1 < r ≤ 3 → punto fisso r = 4 → caos pieno
δ_F = 4.6692... (costante di Feigenbaum, universale)
Teorema di Šarkovskij: l'esistenza di un'orbita di periodo 3 implica orbite di tutti i periodi. La finestra di periodo 3 (r ≈ 3.83) è dunque seguita necessariamente dal caos.

Perché il doppio pendolo non ha attrattore

Il teorema di Liouville garantisce la conservazione del volume nello spazio delle fasi hamiltoniano. Non può esistere una struttura attrattrice di dimensione inferiore: le traiettorie si distribuiscono sulla superficie di energia costante senza convergere verso nulla.

Lo zero nello spettro di Lyapunov

Nei sistemi continui (flussi) la direzione tangente all'orbita ha sempre λ = 0: perturbazioni nella direzione del flusso non crescono né decrescono. Lo spettro del sistema di Lorenz è quindi {+, 0, −} e quello del doppio pendolo (4D) contiene due zeri.

Caos e termodinamica

Il caos hamiltoniano è il meccanismo microscopico che giustifica l'ipotesi ergodica di Boltzmann. Il caos dissipativo mostra come sistemi aperti lontano dall'equilibrio organizzino strutture stabili — ponte verso le strutture dissipative di Prigogine (Nobel 1977).

Universalità: la costante δ_F = 4.6692... è stata misurata sperimentalmente in circuiti elettronici (Linsay, 1981) e instabilità idrodinamiche (Libchaber, Nobel 1996). È una delle poche previsioni quantitative universali del caos.
